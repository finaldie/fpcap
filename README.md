# fpcap
A library for analysis the tcp traffic (based on the libpcap)

## Compile and Install
### Compile the dependency
For example, in ubuntu:<br>
```
apt-get install libpcap0.8 libpcap0.8-dev
make dep
```

### Compile
```
make
```

### Installation
```
make install
```
**notes:** by default, it will be installed to `/usr/local`
